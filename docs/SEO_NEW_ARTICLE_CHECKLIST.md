# 新文章 SEO Checklist（每篇都跑一次）

> 任何新文章發布、Meta 改寫、Landing page 生成前，**逐項打勾**。
> 來源原則皆延續 `SEO_CONTEXT.md` 與 `SEO_STRATEGY.md`，本檔只描述「動作清單」。

---

## A. 動筆前

- [ ] **目標關鍵字**：1 個主要 + 1–2 個次要長尾，寫進文章 frontmatter
  - 主要詞必須通過：包含「地點（北屯/台中/昌平路）」**或** 包含「服務（中藥調劑/慢箋/{症狀}）」
  - 不打全國競爭過強的字（「藥局」「保健食品」單字 → 必加地點）
- [ ] **搜尋意圖**標示為：`資訊型 / 比較型 / 交易型`
- [ ] **對手檢索**：把主要關鍵字丟 Google，記錄前 5 名來源，標示空缺
- [ ] **連到的商品**：列出本文要引用到的 ≥3 個商品頁 URL
- [ ] **連到的服務**：標出對應的 Bundle（見 `SEO_STRATEGY.md` 第 2 段）

## B. 寫稿時

- [ ] **H1** 含主要關鍵字、字數 < 30 字
- [ ] **首段 100 字內**重複主要關鍵字 1 次、提及「北屯昌平路 / 惠登藥局」1 次
- [ ] **H2** 至少 3 段，其中 1 個含次要長尾
- [ ] **段落**：每段 < 4 行；50+ 客群閱讀友善
- [ ] **內部連結** ≥3 個（→ 商品頁 / 其他文章 / 服務頁）
- [ ] **外部連結** ≥1 個（衛生福利部、藥師公會、學術期刊）— 提升 EEAT
- [ ] **CTA**：文末 1 個 LINE 諮詢、1 個 Google Maps 導航 / 來電
- [ ] **圖片**：≥1 張、`alt` 含主要關鍵字、檔名含主要關鍵字（中英皆可）
- [ ] **法規語氣**：避免絕對療效宣稱；用「諮詢藥師」「建議用法」「依專業判斷」

## C. Meta / 技術

- [ ] **`<title>`**：`{文章標題}｜惠登藥局 北屯昌平路` ≤ 60 字
- [ ] **`<meta name="description">`**：含主要關鍵字、地點、電話 ≤ 155 字
- [ ] **Canonical**：指向本頁絕對網址
- [ ] **OG 三件套**：`og:title` / `og:description` / `og:image`（圖至少 1200×630）
- [ ] **Twitter Card**：`summary_large_image`
- [ ] **`<time datetime="YYYY-MM-DD">`** 顯示在文章開頭
- [ ] **結構化資料**：`Article` schema（含 `author`, `datePublished`, `publisher`, `image`）
- [ ] **加入 sitemap.xml**（透過 `build_data.py` 自動帶入）
- [ ] **內部連結雙向**：被引用的商品頁要回連到本文（最好在「相關內容」區塊）

## D. 發布前

- [ ] `python build_data.py` 跑過、本機 `python -m http.server` 預覽
- [ ] 手機版檢視（已用骨架屏 + 電話鈕 + 地圖，務必能跑）
- [ ] DevTools → Lighthouse Mobile：SEO ≥ 95、Accessibility ≥ 90
- [ ] **Clarity 確認**：URL 上線後送 1 個自己的 session
- [ ] git commit 訊息格式：`新文章：〈{文章標題}〉鎖 {主要關鍵字}`

## E. 發布後 7 天

- [ ] Google Search Console → URL 檢查 → **要求索引**
- [ ] 把文章丟一篇 LINE 廣播給熟客
- [ ] 在 IG 限動 + 貼文同步
- [ ] 14 天後檢查：是否進前 30、CTR、點擊量、Clarity 是否有人讀到底

## F. 對應到既有 docs 的去處

| 寫完之後要更新的檔 | 寫什麼 |
|---|---|
| `docs/SEO_CONTEXT.md` →「最近更新」 | 一行：日期 + 新文章標題 + 主要關鍵字 |
| `docs/SEO_STRATEGY.md` → 內容生產順序 | 打勾、改成「已發」 |
| `docs/competitors/*.md` →「關鍵字觀察」 | 若發現對手有新文章 / 新分類，記下來 |

## 反例（不要做的事）

- ❌ 「藥局推薦 TOP 10」這種農場文 — 我們是商家本身，沒立場排他人
- ❌ 標題寫「最便宜」「保證有效」 — 違反藥事法廣告規範
- ❌ 用 GenAI 直翻國外文章未在地化 — 信任度為 0
- ❌ 把同一段話塞到很多商品頁的 description（會被視為 doorway page）
- ❌ 內文塞「北屯北屯北屯」關鍵字堆疊 — 自然出現 2–3 次就好
